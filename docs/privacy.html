ClaWire Privacy Policy

Effective date: 2026-03-01

1. 개요 (Overview)

ClaWire는 OpenClaw용 오픈소스 플러그인입니다. 앱이 수집하는 모든 데이터는 사용자 본인이 직접 운영하는 ClaWire 서버로만 전송됩니다. ClaWire is an open-source plugin for OpenClaw. All data collected by the app is transmitted only to the ClaWire server operated by the user.

2. 수집하는 데이터 (Data Collected)

ClaWire는 다음 정보를 처리할 수 있습니다. ClaWire may process the following categories of data:

  • 위치 정보 (Location)
  • 배터리 상태 (Battery status)
  • 캘린더 이벤트 제목 및 시간 (Calendar event titles and times)
  • WiFi 연결 상태 (WiFi connection status)
  • 앱 사용 상태 (App usage state)

3. 데이터 전송 (Data Transmission)

모든 데이터는 사용자가 직접 설치하고 운영하는 ClaWire 서버로만 전송됩니다. 제3자 서버, 외부 광고 네트워크, 분석 서비스 또는 클라우드 제공업체로 전송되지 않습니다. All data is sent only to the ClaWire server that the user installs and operates. No data is transmitted to third-party servers, advertising networks, analytics services, or cloud providers.

4. 데이터 저장 (Storage)

데이터는 사용자 서버의 SQLite 데이터베이스에 로컬로 저장됩니다. ClaWire 자체는 별도의 클라우드 저장소를 제공하지 않습니다. Data is stored locally in the SQLite database on the user's own server. ClaWire does not provide separate cloud storage.

5. 연락처 (Contact)

문의 또는 개인정보 관련 요청은 GitHub Issues를 이용해 주세요: For questions or privacy-related requests, please use GitHub Issues: https://github.com/paperbags1103-hash/openmantis/issues
