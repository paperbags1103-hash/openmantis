## Description (Korean, 4000 chars max)
ClaWire는 iPhone의 신호(위치, 배터리, 캘린더, WiFi)를 AI 어시스턴트에게 연결하는 OpenClaw 플러그인입니다.

**주요 기능**
• 집/회사 도착·출발 자동 감지 → AI 맞춤 알림
• 배터리 부족 시 일정 기반 맞춤 안내
• 30분 전 일정 사전 알림
• WiFi 전환 감지 (집 ↔ 외부)

**완전한 프라이버시**
모든 데이터는 사용자 본인이 직접 운영하는 서버에만 전송됩니다. 제3자 클라우드 서비스로 데이터가 전송되지 않습니다.

**OpenClaw 필요**
이 앱은 OpenClaw(openclaw.ai)가 설치된 Mac, Linux 또는 Windows 컴퓨터가 필요합니다. OpenClaw는 개인 AI 어시스턴트 플랫폼입니다.

**설치 방법**
1. OpenClaw가 설치된 컴퓨터에서 install.sh 실행
2. ClaWire 앱에서 QR코드 스캔
3. 권한 허용 후 연결 완료

## Description (English)
ClaWire is an OpenClaw plugin that connects your iPhone's signals (location, battery, calendar, WiFi) to your personal AI assistant.

**Features**
• Auto-detect home/office arrivals and departures → personalized AI notifications
• Low battery alerts based on your schedule
• 30-minute calendar event reminders
• WiFi network change detection

**Complete Privacy**
All data is sent exclusively to your own self-hosted ClaWire server. No data is transmitted to third-party cloud services.

**Requires OpenClaw**
This app requires a Mac, Linux, or Windows computer running OpenClaw (openclaw.ai) — a personal AI assistant platform.
